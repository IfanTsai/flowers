## 文件夹说明
- exe/flowers.exe 为编译好可直接在windows平台下运行的可执行程序
- mp4/
  - screen_recording.gif为运行效果的gif
  - screen_recording.mp4为运行效果的mp4(包括bgm)
  - screenshot.png为运行效果的启动截屏
- images/ mp3/ 为该工程需要用到的素材
